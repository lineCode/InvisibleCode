InvisiableCode备注
=================

该项目的初衷是参照QR码设计了一种彩色二维码，希望结合颜色信息用来定位。

关于Visual Studio 2012项目说明
=============================

1、筛选器3DCode下的文件来自于项目3DCode

2、筛选器UI显示当前项目的UI文件

3、该项目引用了来自3DCode项目的文件，需要将此项目置于本项目的同一级目录
